COVID-19 and Mobility:

An investigation of the relationship between COVID-19 and Mobility

Predicting COVID-19 Cases and Deaths based on Mobility using machine learning and data visualization. Using mobility data and Covid-19 case data, we have investigated the relationship between Covid-19 cases, deaths and mobility in various counties. We have also created a multi-output regressor model that predicts the future number of covid cases and number of deaths in the following weeks based on the past covid data and the mobility data, with 90.6% accuracy for cases in the next week.

Health Bill Estimator

"HBE" is a web app where people can input their information into a form and using data analytics, an estimated cost of the hospital bill will be displayed. The app will also enable users to input actual hospital bill cost, this will enable us to collect data useful for updating our dataset, so we can improve our prediction.
